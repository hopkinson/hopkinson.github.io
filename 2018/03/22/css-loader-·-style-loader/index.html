css-loader · style-loader

loaders 含义
css-loader 解析css文件，通过import加载，返回css
style-loader 通过添加<style>把css添加到DOM

建议将 style-loader 与 css-loader 结合使用

用法：
1.开发环境中，加载css文件

2.生产环境中，通过extract-text-webpack-plugin从bundle提取css，实现并行加载css资源
